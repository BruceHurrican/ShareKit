# 关于 ShareKit

从 https://github.com/maquanwei/ShareKit.git 继续开发，旨在剥离UI和接口，是开发者更加方便调用。之前的ShareKit包含了许多UI部分，而且每一个平台的分享单独进行。这次进行了一个整合，可以一键分享新浪，豆瓣，腾讯三个平台，支持图片和文字一齐分享。


# 服务

## 可用服务

1. 新浪微博
3. 腾讯微博
4. 豆瓣
